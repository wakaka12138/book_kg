# book_kg
我们对教科书语步识别方法进行了研究，以《计算机科学概论》书中第二章内容为样例，应用教科书语步识别方法自动生成了教科书知识图谱。
## 数据来源
《计算机科学概论》（第12版） 第二章：数据操控
